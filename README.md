# valuation_preco_teto


Link do repo: [https://github.com/kayohr/valuation_preco_teto](https://valuation-preco-teto.onrender.com/)

Este projeto foi desenvolvido com o objetivo de facilitar a visualização, organização e acessibilidade de análises originalmente construídas em Jupyter Notebook.

A proposta é transformar notebooks em uma estrutura mais amigável para leitura, estudo e compartilhamento.
